# KOSPI 대형주 베타, 모멘텀 전략(25.12.02)
## 1. 프로젝트 개요
이 프로젝트는 코스피 시가총액 상위 50개 종목을 대상으로 베타와 모멘텀 팩터를 결합하여, 
예상되는 시장 하락에서 안정적으로 초과 수익을 추구하는 퀀트 전략을 구현하고 검증한 프로젝트입니다.

## 2. 퀀트 전략
* 로우 KOSPI 시가총액 상위 50위 (대형주)
* 팩터
  * 베타
  * 모멘텀(1년)
* 운용 방식 : 월간 리밸런싱

## 3. 결과
* 기간 : 2021.01 ~ 2025.12.02
* 총 수익률 : **349.66%** (vs KOSPI 29.68%)
* 초과 수익률 : **+319.98%p**
